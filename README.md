# Weather

This is a simple Jetpack Compose weather app for Android.
Weather data is provided by [OpenMeteo](https://open-meteo.com/en/licence).
The app uses [Meteocons](https://meteocons.com/) weather icons.

### Compatability
- Android API level 33+ is required since background updates utilize the WorkManager API

### Features
- Current, hourly and daily forecasts
- Severe weather alert notifications
- Forecasts based on either device location or a user-specified location
